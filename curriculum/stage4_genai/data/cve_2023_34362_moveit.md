# CVE-2023-34362 — MOVEit Transfer SQL Injection

**CVSS:** 9.8 | **Severity:** Critical | **Disclosed:** 2023-06-01

## Summary
CVE-2023-34362 is a critical SQL injection vulnerability in Progress Software's MOVEit Transfer managed file transfer application. An unauthenticated attacker can submit crafted SQL statements to the MOVEit Transfer web application, bypassing authentication and gaining access to the underlying database. This enables the attacker to extract sensitive data, modify database contents, and achieve remote code execution by abusing database-level functionality. The Cl0p ransomware group mass-exploited this vulnerability to exfiltrate data from hundreds of organizations.

## Affected Products
- MOVEit Transfer 2023.0.0 (15.0) and earlier
- MOVEit Transfer 2022.1.x (14.1) prior to 2022.1.6
- MOVEit Transfer 2022.0.x (14.0) prior to 2022.0.7
- MOVEit Transfer 2021.1.x (13.1) prior to 2021.1.7
- MOVEit Transfer 2021.0.x (13.0) prior to 2021.0.9
- MOVEit Cloud was also affected and patched by Progress

## Attack Vector
The attacker sends a crafted HTTP request to the `/moveitisapi/moveitisapi.dll` or `/guestaccess.aspx` endpoint containing a SQL injection payload in a session token or request parameter. The injected SQL queries the MOVEit database (Microsoft SQL Server) to extract an API access token or sysadmin session, then leverages the `xp_cmdshell` stored procedure or deserialization of stored data to execute operating system commands. The attacker uses this access to deploy a webshell (commonly named `human2.aspx`) in the MOVEit web directory for persistent access and bulk data exfiltration.

## Detection
- Search IIS logs for requests to `/moveitisapi/moveitisapi.dll` with unusual query strings or large POST bodies
- Look for unexpected files in the MOVEit installation directory, particularly `human2.aspx` or similarly named `.aspx` files
- Monitor SQL Server logs for `xp_cmdshell` execution or unusual query patterns from the MOVEit application account
- Network: large outbound data transfers from the MOVEit server to unfamiliar external IP addresses

## Remediation
- **Patch:** Apply the MOVEit Transfer security patch released June 2023 (DLL drop-in patch for each affected version) or upgrade to MOVEit Transfer 2023.0.2+
- **Workaround:** Disable all HTTP and HTTPS traffic to the MOVEit Transfer environment until the patch is applied; review and remove unauthorized accounts or webshells
