# Facilitator Guides

Standardised teacher notes for each lecture in the AI Ninja Program. Each guide follows the same template so any facilitator — whether the program lead or a Tier 3 graduate — can deliver a consistent, high-quality session.

## How to Use These Guides

1. **Read the guide** before the live session (15 min prep)
2. **Open the matching PPTX** — slide numbers in the guide map to the deck
3. **Follow the timing** — each section has a suggested duration to keep you on track
4. **Use the discussion prompts** — these drive the conversations that make live sessions valuable
5. **Review "Common Questions"** — participants will ask these; having answers ready builds credibility
