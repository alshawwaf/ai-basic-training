Lab: Lakera-Demo

A web-based testing platform with 24+ documented attack vectors across 9 categories.

git clone https://github.com/alshawwaf/Lakera-Demo.git
cd Lakera-Demo
python -m venv venv
# Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# Edit .env — add your LAKERA_API_KEY and LLM provider key
python app.py

Access at http://127.0.0.1:9000

Exercise 1: Test the Attack Library (20 min)

Open the Playground and test at least 5 attacks from different categories:

  • Jailbreak (role-play bypass)
  • Prompt injection (instruction override)
  • Data extraction (system prompt leak)
  • PII extraction
  • Toxic content generation

For each: note whether it was caught, the category assigned, and the confidence score.

Exercise 2: Craft a Novel Attack (15 min)

Write your own attack prompt that is NOT in the built-in library. Try:

  • Encoding the attack in a different language
  • Splitting the malicious instruction across multiple turns
  • Embedding the attack in a "summarise this document" request

Did the guardrails catch your custom attack? Why or why not?

Exercise 3: Benchmark (10 min)

Use the benchmarking feature to compare detection rates. Which attack categories have the highest detection rate? The lowest? If building multi-layered defense, which providers would you combine?
