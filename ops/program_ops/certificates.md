# Ninja Program — Certificate Templates

> Design specifications for the three certification tiers. Each certificate is a single-page landscape document (11" x 8.5" or A4 landscape).

---

## Common Elements (all tiers)

| Element | Specification |
|---------|--------------|
| Orientation | Landscape |
| Font | Calibri |
| Border | Thin line border with tier-specific accent colour |
| Logo | Company logo top-left |
| Program name | "AI Ninja Program" — centred, large |
| Recipient line | "This certifies that [NAME]" |
| Achievement line | Tier-specific (see below) |
| Date line | "Completed [DATE]" |
| Signatures | Program Tech Lead + Engineering Manager |
| Certificate ID | Unique ID format: NINJA-[TIER]-[COHORT]-[NUMBER] (e.g., NINJA-T3-C01-007) |
