Context vectors: attention in action

The attention weights determine how much each word contributes to the context vector — a new representation that captures meaning from surrounding words.

Why this matters

Without attention: "blocked" would have the same representation regardless of context. "Firewall blocked traffic" = "Road blocked ahead" = same vector. Meaningless!
With attention: "blocked" in "firewall blocked malicious" gets a context vector enriched by "firewall" and "malicious" — the model knows it's a security action.

Security connection: Context vectors are why LLMs can distinguish "port 443 is open" (network status) from "please open the attachment" (phishing). The word "open" gets different context vectors depending on surrounding tokens.
