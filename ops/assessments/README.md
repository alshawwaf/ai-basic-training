# Assessments

Assessment gates between stages verify understanding and qualify participants for the next tier.

| Gate | After | Format | Duration | Passing |
|------|-------|--------|----------|---------|
| Gate 1 | Stages 0-1 | Quiz (10 questions) + code challenge | 60 min | 70% quiz + working code |
| Gate 2 | Stage 2 | Timed mini-project on a new dataset | 60 min | Working pipeline + correct metrics |
| Gate 3 | Stage 3 | Architecture review (identify issues in a diagram) | 45 min | 70% of issues found |
| Gate 4 | Stage 4 | Live capstone demo to a review panel | 15 min | Rubric score 3/5 or higher |

## Tier Certification

| Tier | Requires | Title |
|------|----------|-------|
| Tier 1 — AI Foundations | Gate 1 + Gate 2 | AI Foundations Certified |
| Tier 2 — AI Practitioner | Tier 1 + Gate 3 | AI Practitioner Certified |
| Tier 3 — AI Ninja | Tier 2 + Gate 4 | AI Ninja |
