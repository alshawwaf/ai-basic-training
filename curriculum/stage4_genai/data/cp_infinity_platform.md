# Check Point Infinity Platform

## Overview

The Check Point Infinity Platform is a unified, AI-powered security platform that consolidates network security, cloud security, workspace security, and AI security under a single management and intelligence layer. Rather than deploying disconnected point products, Infinity provides a consolidated architecture where every enforcement point shares threat intelligence, policy, and management through a common backbone.

## Platform Pillars

| Pillar | Scope | Key Products |
|--------|-------|-------------|
| Hybrid Mesh Network Security | On-premises and cloud network perimeter | Quantum Security Gateways, SD-WAN, Cloud Firewalls, WAF |
| Workspace Security | Endpoints, email, and remote access | Harmony Endpoint, Harmony Email & Collaboration, Harmony SASE, Harmony Browse |
| Exposure Management | Attack surface visibility and risk reduction | External Risk Management, Threat Exposure Management |
| AI Security | Governance and protection of AI tools and applications | Workforce AI Security, AI Agent Security, AI Guardrails |

## Management Layer

| Component | Function |
|-----------|----------|
| Management & Smart-1 Cloud | Centralised policy management, logging, and reporting for all enforcement points |
| Spark Management | Simplified management designed for small and medium business deployments |
| XDR / Infinity XDR | Cross-domain detection and response — correlates events from network, endpoint, email, and cloud |

## ThreatCloud AI — The Intelligence Backbone

ThreatCloud AI is the threat intelligence engine that powers every enforcement point in the Infinity Platform. It aggregates threat data from all Check Point sensors globally — firewalls, endpoints, mobile devices, cloud workloads, and email gateways — and uses machine learning models to generate real-time threat verdicts. Every gateway, agent, and cloud instance in the Infinity ecosystem receives continuous updates from ThreatCloud AI, ensuring that a threat detected at one enforcement point is immediately blocked across all others.

## AI Security Suite

The AI Security suite extends the Infinity Platform to address the risks introduced by organisational AI adoption. Workforce AI Security governs employee AI tool usage. AI Agent Security monitors autonomous AI agents and their tool access. AI Guardrails protects LLM-powered applications against adversarial prompts and data leakage. Together, these three products provide a comprehensive AI governance and protection layer integrated into the broader Infinity architecture.
