A Large Language Model is a next-token predictor

LLM = Large Language Model — the engine behind ChatGPT, Claude, and Gemini. Despite the name, its job is surprisingly narrow:

Given the words so far, predict what word comes next — one token at a time.

Everything else — reasoning, coding, summarisation — emerges from running this loop billions of times on trillions of words.

Predicted next tokens
The loop: predict → append → predict → append. Each token is chosen based on all previous tokens.

Cybersecurity tie-in: When a security analyst asks an LLM to "analyze this log entry", the model predicts the most likely next tokens given the context. It doesn't "understand" the log — it generates the most probable continuation based on patterns learned from training data.
